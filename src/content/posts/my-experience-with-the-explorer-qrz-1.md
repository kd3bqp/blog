While it's not usually the first radio I grab — that honor goes to my BTech UV-PRO — the Explorer QRZ-1 is a genuinely interesting handheld, and I've been living with it long enough to have opinions.

The SDR-based design does some clever things, and the wideband receive is a lot of fun to scan with. The interface takes some getting used to and the software still feels young, but you can tell where it's headed.

If you like tinkering and being a little early to a platform, it's worth a look. If you want a grab-and-go that just works, I'd still start elsewhere.
